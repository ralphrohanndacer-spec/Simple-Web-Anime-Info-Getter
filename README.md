# Anime Info Getter

A simple web application that searches anime information using the Kitsu REST API.

**Features**
- Search anime by title
- Filter results by TV/MOVIE/OVA
- Displays:
  - Poster image
  - English title
  - Japanese title
  - Release date
  - Rating/Score
  - Num of episodes
  - Status
  - MAL ID
  - Synopsis
- Dynamic anime background
- Poster image url included
- Clickable poster image

**Built With**
- HTML
- CSS
- JavaScript
- Kitsu REST API
